The Funny Rooms

The funny rooms is a game made by artists: Vic Ramjiawansingh, Thierry van Rooijen & Zoe Snijder. And developers: Damian Ockers, Xander Kortekaas & Fabian Dekker. The theme for this year was: Make me laugh.

Sadly, we could not think of a good game idea that fit the theme of this year. So we decided there were going to be 3 rooms with all different scenarios to make the player laugh. I made the right room, in which you had to solve a color puzzle while killing snakes. The idea for this room was to make it scary/funny, but there was not enough time to make that work.

For this room I made a snake spawner in which new snakes spawn every wave, for 3 waves. As the player, you have to grab a weapon and kill the snakes. For the button wall, the player has to remember the locations the snakes spawned in and only hit those buttons. After clearing all the waves and solving the colored button puzzle, the player can leave the room.

Turnpip seeks new ground

This was the first time I participated in a Game Jam. The theme was roots. I made this game together with Damian Ockers in which he made almost all the code and I did almost all the art. We could not find an artist in time for this game, so we decided that I was going to make the art for the game. I made the art in Aseprite and designed some simple characters and a tileset. We were happy with how the game came out, but wished we had more time or people to finalize the game.
