Dread Race

Dread Race is a race game in which you try to complete 3 laps as fast as possible!

The track is made up out of lots of twists and turns, and a split path as soon as you exit the church.

My Contributions:

Drifting

Because the track has many sharp turns, a drift was absolutely necessary. But making the drift work was rather difficult. I needed to make sure you could not drift for too long, as you would just spin out, and in needed to feel good when you use it. So, I decided to use two sets of wheels. The first set is for driving normally, the second one is for when the player decides to drift.

This way I could give new values to the wheels without having to change them back. For instance, the normal wheels forward- and sideways frictions asymptote slip sits at a 0.8f and 0.05f respectfully. While for the drift wheels the same forward- and sideways frictions asymptote slip is a 0.25f and 0.9f respectfully. What this means is that while the drift is engaged the cars forward friction is set lower than the sideways friction, so that the car turns sideways and drifts.

Path Triggers

Because the track has a split path, there needed to be restrictions on the other path so the player would not drive around in circles. This is why, when you exit the church and choose a path (let's say path A), the other path (path B) gets blocked off. The closed path (path B) only opens again when the player chooses that path (path B) when it exits the church, of course this means that again the other path (path A) gets closed off.

Learn more about my code here


Checkpoints

Just in case the player got stuck off the road or does not remember which way to go, there needed to be a checkpoint system. So, over the track are some checkpoints, whenever the player drives into one of these checkpoints, their position and rotation gets saved. This way, whenever the player presses the respawn key, they get teleported back towards the last touched checkpoint.
